
# Exam Scheduler

An automization for scheduling university exams.


## Run

Clone the project

```bash
  git clone https://github.com/anovius/Exam-Scheduler.git
```

Go to the project directory

```bash
  cd Exam-Scheduler
```

Install dependencies for frontend

```bash
  cd client
  npm install
```

Start the client

```bash
  npm run start
  goto http://localhost:3000
```

  
## Tech Stack
**MERN Stack**

**Client:** React JS

**Server:** Node, Express

**Database:** MongoDB
